This is my first mod! It's rather simple and aims to recreate a mod i vividly remember from mcpe.

It's features are:
- A custom Speed Block that gives you Speed III when stepping on it
- A custom Slime Stick that shortly (1s) turns a clicked block into a slime block and then back
- A custom Dash Stick that allows you to dash into the direction you're looking at (5s cooldown)

With those features you can create some fun parkours and generally fly around and clutch in your world :)
